# 请用于合法用途

使用TAP模式（为游戏优化）

无NAT和DNS，仅保留局域网功能

## 使用方法

第一步，下载脚本：

```bash
curl -O https://raw.githubusercontent.com/Nouko61/openvpn-install/master/openvpn-install.sh
chmod +x openvpn-install.sh
```

第二步，运行：

```sh
./openvpn-install.sh
```
